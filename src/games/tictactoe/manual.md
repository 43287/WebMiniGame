# 基本规则 (普通模式)

经典的井字棋游戏。

- 棋盘为 3x3 的网格。
- 两名玩家轮流落子（X 和 O）。
- 最先在横、竖、斜方向连成一线者获胜。
- 若棋盘下满仍无连线，则为平局。

# 超级井字棋 (Super Mode)

这是一个极具策略深度的变体，棋盘由 9 个小的 3x3 网格组成一个大的 3x3 网格（共 81 格）。

### 🎯 落子约束
你在小棋盘中的落子位置，决定了对手下一步**必须**在哪个大棋盘中落子。
> 例如：如果你在当前小格的“右下角”落子，对手下一步只能在整个棋盘的“右下角”那个大格里下。

### 🏆 占领大格
当你在一个小棋盘中获胜（连成一线），你就占领了这个大格。该区域会被标记为你的符号（大号 X 或 O）。

### 🔓 自由落子
如果对手把你送到了一个已经占领（或已满平局）的大格，你将获得**自由落子权**，可以在任意未满的大格中落子！这通常是扭转局势的关键。

# 覆盖井字棋 (Cover Mode)

一种增加了“升级”维度的玩法，允许覆盖已有的棋子。

- **小标志**：普通棋子。点击空位放置。
- **大标志**：高级棋子。每人拥有 **3个**。
- **覆盖规则**：当一个格子上已经有**小标志**（无论是谁的）时，你可以消耗一个大标志将其**覆盖**。
- **不可逆**：大标志一旦放置，就不能再被覆盖。
- **胜负**：只要当前最上层的标志连成一线，即获胜。

> 策略提示：谨慎使用有限的大标志，它们是逆转局势的关键！

# 胜利条件

- **首要条件：**在大棋盘上，占领的大格连成一线（横、竖、斜）。
- **次要条件：**如果大棋盘全部占领或下满且未连成一线，则占领大格数量多者获胜。
