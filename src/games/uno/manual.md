# 基本规则

UNO 是一种经典的卡牌游戏。目标是最先打完手中的牌。

- 每位玩家开局获得 7 张牌。
- 按顺时针方向轮流出牌。
- 出的牌必须与上一张牌在 **颜色** 或 **数字/符号** 上匹配。
- 如果无牌可出，必须从牌堆摸一张牌。

# 特殊功能牌

- **🚫 跳过 (Skip)**：下家停止出牌一次。
- **🔁 反转 (Reverse)**：改变出牌方向（顺时针 ↔ 逆时针）。
- **+2 (Draw Two)**：下家摸两张牌并跳过回合。
- **🌈 变色 (Wild)**：可以随时打出，并指定下一轮的颜色。
- **+4 (Wild Draw Four)**：指定颜色，下家摸四张并跳过。如果不被质疑成功的话。

# 进阶规则说明

**叠加惩罚 (Stacking)**
当上家打出 +2 时，你可以打出另一张 +2 将惩罚转移给下家（此时变为 +4）。
若开启 "**+4 叠加 +2**"，则可以用 +4 回应 +2。

**抢牌 (Jump-In)**
如果你手中有与刚才打出的牌 **完全相同**（颜色+数字/符号）的牌，可以无视回合顺序直接打出，抢夺出牌权。

**7-0 规则**
- 打出 **7**：可选择一名玩家交换全部手牌。
- 打出 **0**：所有玩家手牌按出牌方向轮转。

**质疑 (Challenge)**
当有人对你打出 +4 时，如果你认为他手中有当前颜色的牌（即他不应该用 +4），你可以发起质疑。
- 成功：他自己摸 4 张。
- 失败：你摸 6 张（原4张+2张罚牌）。

# 胜利条件

当打出倒数第二张牌时，必须喊 **UNO!**（系统会自动检测或提供按钮）。
最先打完手中所有牌的玩家获胜。
